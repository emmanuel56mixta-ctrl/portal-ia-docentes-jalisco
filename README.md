# Portal IA para Docentes · Jalisco

Portal del **Constructor pedagógico**: herramienta digital con inteligencia artificial para elaborar y organizar el trabajo docente, impulsada por Google Gemini. Incluye la ruta del diagnóstico al aula, el banco de Gemas especializadas, cursos y centro de ayuda.

## Páginas principales
- `academia-docente-jalisco-v2.html` — Página principal
- `ruta-v2.html` — Constructor pedagógico (módulos de diagnóstico, programa analítico, planeación y evaluación)
- `banco-gemas-v2.html` — Banco de Gemas especializadas
- `gemas-v2.html` — Banco de recursos
- `ayuda-ruta.html` — Centro de ayuda
- `curso-herramientas-ia-v2.html`, `curso-primeros-pasos-v2.html`, `leccion-1-v2.html`, `recuperar-cuenta-v2.html`
